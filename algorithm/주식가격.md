# 주식가격
## 문제 설명
* 입력: 초 단위로 기록된 주식가격 배열 prices
* 출력: 가격이 떨어지지 않은 기간(초) 배열

## 제한사항
* 1 <= prices[n] <= 10000
* 2 <= len(prices) <= 1000000

## 입출력 예
|prices|return|
|------|------|
|[1, 2, 3, 2, 3]|[4, 3, 1, 1, 0]|

```python
# Brutal Force
def solution(prices):
    answer = [0] * len(prices)
    idx = 0
    while prices:
        pivot = prices.pop(0)
        for price in prices:
            answer[idx] += 1
            if pivot > price:
                break
        idx += 1
    return answer
```
